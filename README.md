# Qubit Calculus — Agency Website

The marketing site for [Qubit Calculus](https://qubitcalculus.com) — a boutique software development agency specializing in high-performance, real-time web applications.

## Quick Start

```bash
npm install
npm run dev     # http://localhost:3001
```

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** for blazing-fast bundling
- **Tailwind CSS** for styling
- **Framer Motion** + **GSAP** for premium animations
- **React Router** for SPA navigation
- **React Helmet Async** for SEO

## Build

```bash
npm run build    # Production bundle → dist/
npm run preview  # Preview production build
```
